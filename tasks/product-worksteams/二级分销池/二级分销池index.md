# 二级分销池 Index

## 项目目标

二级分销池项目用于支持渠道方独立运营事件合约与足球预测市场站点，同时复用 TurboFlow 底层报价、交易、池子、结算与风控能力。

v2.3 的核心升级是：渠道站用户与 TurboFlow 主站用户隔离，渠道资金与主站资金隔离，并在两个产品上统一引入渠道报价 markup、渠道收入对账与足球预测市场分销池路由能力。

## 核心范围

- 渠道独立域名、独立前端、独立用户入口。
- 渠道用户打 `channel_id` / `identity_domain`，与 TurboFlow 主站用户不可跨登录。
- 渠道余额按 `fund_domain` 与主站余额完全隔离，不可跨充值、提现或复用。
- 渠道 odds quote 在 TurboFlow base quote / future best_quote 上叠加 markup p%。
- 事件合约 v2.3：渠道订单仍 100% 进入 TF 公共池。
- 足球预测市场 v2.3：支持渠道分销池路由开关。
- markup 收入、渠道池盈亏、充值、提现、cashbook 可按渠道对账。
