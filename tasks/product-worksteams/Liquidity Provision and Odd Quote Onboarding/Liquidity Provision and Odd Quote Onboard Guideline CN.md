# 流动性提供与赔率报价接入指南

**状态：** Ready for review  
**最后更新：** 2026-05-25 18:38 +08  

## 1. 如何设置流动性池

1. 与 TurboFlow 确认产品范围：
   - 事件合约 / 猜涨跌
   - 足球预测市场
   - 或两者都支持

2. 创建或确认用于持有 / 运营流动性池的 TurboFlow 账户。

3. 完成资金准备：
   - 确认注资资产。
   - 确认初始池子金额。
   - 确认结算 / 运营钱包。
   - 确认风险限额或最大敞口。

4. 向 TurboFlow 提供账户标识：
   - UID / account ID
   - 如适用，提供**钱包地址**
   - 支持的产品与市场列表

5. TurboFlow 创建专属流动性池，并返回：
   - **`pool_id`**
   - 支持市场
   - 如适用，链上 / 池子监控信息
   - 如适用，结算与对账信息

6. 满足以下条件后，TurboFlow 开启池子：
   - 资金确认到账。
   - 产品与市场权限已配置。
   - 池子风险限额已配置。
   - 如池子依赖外部报价，报价 API 已准备好。

## 2. 如何开始赔率报价

报价接入请直接使用对应产品的 API 文档。TurboFlow 会在接入时提供环境 host、凭据与允许市场列表。

| 产品 | 报价范围 | API 文档 |
| --- | --- | --- |
| 事件合约 / 猜涨跌 | 按 market 推送 `up` / `down` 的 `return_rate`，例如 `BTC-5m` | [事件合约 v2.2 赔率报价 API](../事件合约（猜涨跌）/v2.2%20-%20多vendor流动性路由方案/API文档%20-%20多vendor流动性路由方案%20v2.2%20CN.md) |
| 足球预测市场 | 按 league、比赛 / 系列赛市场、outcome 推送足球欧式赔率 | [足球预测市场 v2.2 赔率报价 API](足球预测市场做市商赔率报价接口%20v2.2%20CN.md) |

开始推送报价前，请与 TurboFlow 确认：

- API 凭据与环境。
- 允许推送的 markets / leagues / outcomes。
- 报价推送频率与过期规则。
- 心跳与重连规则。
- 是否开通 `best_quote` 订阅。
